# Sniffdiff

A work in progress, with failing tests and everything!

###Dependencies:
* Ruby 1.9.2+
* [rubygems](http://rubyforge.org/frs/?group_id=126)
* `gem install sinatra`
* `gem install sinatra-contrib`

### Running tests
```
cd test
ruby test.rb
```

### Starting the app 

`ruby sniffdiff.rb`
